Разработчик проекта

Данный проект создан в рамках отборочного проекта для nFactorial School. Тематика проекта посвящена моей любимой игре — Tekken 7. Основная цель — продемонстрировать навыки работы с современными фронтенд и бэкенд технологиями.

Email: abzaltaaabzal@gmail.com

Выбор технологий

Проект построен с использованием следующего стека:

  • Angular — для создания удобного, модульного и масштабируемого фронтенда. Был выбран благодаря предыдущему опыту работы с этим фреймворком.
  • Django — надёжный и быстрый фреймворк для разработки серверной части (backend API). Принят из-за имеющегося опыта и высокой скорости разработки.
  • Gemini API — используется для интеграции генеративных возможностей и расширения функциональности проекта.

Выбор этих технологий основан на личном опыте и потребности в гибкости, быстроте разработки и масштабируемости проекта.
